Para instalar o webpack usando react é preciso que o index.js 
esteja na pasta src
	-> ./src/index.js


Intalação do React
Depois de instalar e configurar o webpack.config.js package.json
	-> npm install react react-dom
	
	
Instalação do Babel para react
	-> npm install @babel/core @babel/preset-react babel-loader --save-dev
	
	
	
	Para simplificar a criação do ambiente de desenvolvimento do react
	Podemos usar um app create react app para facilitar.
	acesse: https://create-react-app.dev/docs/getting-started
	
	-> npx create-react-app my-app
	-> cd my-app
	-> npm start
