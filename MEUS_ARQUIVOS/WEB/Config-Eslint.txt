Instale o plugin Eslint no Vscode


Instalar o Eslint Globalmente
	-> npm install eslint -g
	
	
Iniciar repositorio NPM Local(package.jason)
	-> npm init
	
	
Iniciar o eslint (use popular Airbnb - json)
	-> eslint --init
