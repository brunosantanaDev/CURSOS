
# Iniciar o NPM no projeto
	-> npm init
	
	(gera o arquivo package.json, nesse arquivo é onde são instaladas as dependecias
	como jquery, lodash, typescript, eslint)

-------------------------------------------------------------------------------------------------

# Instalar PACOTE
	-> npm install 
	
	(Caso ja exista o arquivo package.json, esse comando vai instalar todas as dependecias que estiverem definidas nele)
	
-------------------------------------------------------------------------------------------------

# Instala pacote de modo global
	-> npm install eslint -g

-------------------------------------------------------------------------------------------------

# Atualizar Pacote

	-> npm update typescript

-------------------------------------------------------------------------------------------------

# Desinstala Pacote
	-> npm uninstall typescript
