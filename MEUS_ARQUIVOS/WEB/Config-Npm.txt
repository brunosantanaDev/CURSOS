# Iniciar o NPM no projeto
	-> npm init (faz perguntas)
	-> npm init -y (confirma todas as perguntas)
	
	(gera o arquivo package.json, nesse arquivo é onde são instaladas as dependecias como jquery, lodash, typescript, eslint, react)

-------------------------------------------------------------------------------------------------

# Instalar PACOTE
	-> npm install 
	
	(Caso ja exista o arquivo package.json, esse comando vai instalar todas as dependências que estiverem definidas nele)
	
-------------------------------------------------------------------------------------------------

# Instala pacote de modo global
	-> npm install eslint -g

-------------------------------------------------------------------------------------------------

# Atualizar Pacote

	-> npm update typescript

-------------------------------------------------------------------------------------------------

# Desinstala Pacote
	-> npm uninstall typescript
